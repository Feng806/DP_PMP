104% ----- DERIVE emoPwr FROM batPwr STATE -----------------------------------
105% from the given batPwr, derive emoPwr from equation:
106%   emoPwr = batPwr - batPwrLoss,
107% where
108%   batPwr      = batOcv * I, and
109%   batPwrLoss  = I^2 * batRst.

111% lookup battery voltage for the given battery state - batOcvPre

115% can now caculate battery losses:
